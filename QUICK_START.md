## ⚡ Quick Tasks (30 seconds each)

### Add a Review
1. Database → reviews → + Add Row
2. Fill: name, suburb, rating (5), text
3. Save ✓

### Update Phone Number
1. Click phone number on website
2. Edit in sidebar
3. Save ✓

### Update Hours
1. Code → Home.tsx
2. Find "Mon–Fri 8am–5:30pm"
3. Change it
4. Save ✓

### Undo a Mistake
1. More (⋯) → Version History
2. Click earlier version
3. Rollback ✓
